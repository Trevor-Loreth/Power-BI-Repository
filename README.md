# Power BI Repository
This repository contains a collection of Power BI dashboards and reports designed to analyze business performance, uncover insights, and support data-driven decision-making.

## Overview
This repository showcases Power BI projects focused on:

- Sales performance analysis
- Factory performance analysis
- Product-level insights
- Regional and market trends
- KPI tracking and benchmarking
- Returns analysis
- Defect analysis
- Actionable insights

Each report is interactive and designed with slicers, filters, and dynamic visuals to allow flexible exploration of the data.
