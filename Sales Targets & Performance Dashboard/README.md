# Sales Performance Dashboard

## Overview

Hitting a sales target is one thing, understanding *why* you did or didn’t is another. This dashboard focuses on breaking down **Actual Sales vs Sales Targets** across key business dimensions to uncover performance patterns that aren’t obvious from raw data alone.

The report highlights:
- Where targets are consistently exceeded  
- Where performance is lagging  
- How trends evolve over time  

Users can explore these insights interactively by Product, Region, and Time.

> Inspired by a tutorial from Ali Ahmad, with custom enhancements in structure, visuals, and analysis.

## Dataset
**Source:** Provided by Ali Ahmad via GitHub  
**Format:** CSV  
**File Included:** `actual_vs_target_sales_dataset.csv`

#### Contains:
- Sales Amount  
- Sales Target  
- Product Name  
- Region  
- Date fields  

## Features
- **Custom Measures** Created using DAX to calculate performance metrics and comparisons.
- **KPI Cards** Quick summary of overall performance.
- **Slicers** Filter data by product, region, and time.
- **Visual** Tracking sales trends over time and comparing actual vs target sales.
- **Matrix with Conditional Formatting**  
  - 🟢 High Performance  
  - 🟡 Medium Performance  
  - 🔴 Low Performance
